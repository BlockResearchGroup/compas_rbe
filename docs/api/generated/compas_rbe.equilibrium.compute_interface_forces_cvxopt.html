compute_interface_forces_cvxopt

compas_rbe.equilibrium.compute_interface_forces_cvxopt(assembly, friction8=False, mu=0.6, density=1.0, verbose=True, maxiters=1000)

Compute the forces at the interfaces between the blocks of an assembly.

Solve the following optimisation problem:

\[ \begin{align}\begin{aligned}\begin{aligned}\\\begin{split} & \underset{x}{\text{minimise}} & \quad 0.5 \, \mathbf{x}^{T} \mathbf{P} \mathbf{x} + \mathbf{q}^{T} \mathbf{x} \\ & \text{such that} & \quad \mathbf{A} \mathbf{x} = \mathbf{b} \\ & & \quad \mathbf{G} \mathbf{x} = \mathbf{h} \\\end{split}\\\end{aligned}\end{aligned}\end{align} \]
Parameters

  • assembly (Assembly) – The rigid block assembly.

  • friction8 (bool, optional) – Use an eight-sided friction pyramid. Default is False.

  • mu (float, optional) – ?

  • density (float, optional) – Density of the block material. Default is 1.0

  • verbose (bool, optional) – Print information during the execution of the algorithm. Default is False.

  • maxiters (int, optional) – Maximum number of iterations used by the solver. Default is 100.

Returns

None – The assembly is updated in place.

References

The computational procedure for calculating the interface forces is described in detail in [Frick2015]
